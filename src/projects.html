Machine Learning-Driven Digit Classifier Website

Description: Greetings! I'm pleased to share one of the projects I'm particularly proud of – the Machine Learning-Driven Digit Classifier Website. This platform seamlessly marries web development and machine/deep learning, bringing together intricate tech details with a user-friendly design. Venturing into the world of the MNIST dataset, I tapped into TensorFlow to train a Convolutional Neural Network. It’s this very CNN that powers the website's core functionality, transforming hand-drawn digits into recognizable digital figures. The foundation of this project rests on a Flask-based Python backend. This robust framework ensures that as users interact with the website, they're experiencing efficient, real-time digit recognition. Recognizing the increasing shift towards mobile browsing, I designed an adaptable interface utilizing AJAX and HTML Canvas. The goal was to guarantee that users, be it on a desktop or a mobile device, found the drawing process both intuitive and engaging.

Technologies: TensorFlow, Python, Flask, AWS EC2 Ubuntu Server, JavaScript AJAX, HTML Canvas.

MicroCaml Lexer, Parser, and Interpreter

Description: I dove deep into the realms of language processing with this one – my very own MicroCaml lexer, parser, and interpreter. At its heart, the lexer tokenizes code inputs, simplifying it for the parser. The parser then interprets these tokens to create an Abstract Syntax Tree (AST), which serves as the blueprint for the interpreter. Ultimately, it's the interpreter that does the heavy lifting, turning the processed code into actionable operations. It's like watching a well-coordinated ballet, where the parser choreographs and the interpreter dances, making the code come to life. Additionally, I've ensured that the project runs seamlessly in mutop, a Micro-utop version, providing a hands-on playground for MicroCaml expressions.

Technologies: MicroCaml, Lexer, Parser, Interpreter, operation semantics, Unix OS, Micro-utop.
